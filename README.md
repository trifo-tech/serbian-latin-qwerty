# Srpski Latin (QWERTY) - Windows 11 / 10 Layout

Zvanični Windows instalacioni paket za srpsku latiničnu tastaturu sa **QWERTY** rasporedom tastera (slovo **Z** je dole levo, slovo **Y** je gore pored T). 

Sva naša slova (**Š, Đ, Č, Ć, Ž**) ostaju na svojim standardnim mestima, a raspored je idealan za programere, video editore i gejmere koji žele čist QWERTY sistem bez softvera u pozadini (poput PowerToys-a) koji remeti engleski raspored.

Paket je bezbedno generisan preko alata *Microsoft Keyboard Layout Creator (MSKLC) 1.4* u izolovanom Windows Sandbox okruženju.

---

## 🚀 Kako instalirati?

1. Preuzmite kompletan ovaj repozitorijum (Kliknite na zeleno dugme **Code** -> **Download ZIP** i otpakujte ga na računaru).
2. Otvorite otpakovani folder i pokrenite fajl **`setup.exe`**.
3. Sačekajte sekundu da Windows završi instalaciju i kliknite *Close*.
4. **Restartujte računar** (ili se odjavite i ponovo prijavite na Windows).

---

## ⚙️ Podešavanje u Windowsu 11

Nakon restarta, potrebno je da aktivirate novi raspored:
1. Otvorite **Settings** (`Win + I`) -> **Time & language** -> **Language & region**.
2. Pored jezika *Srpski / Serbian* klikni na tri tačkice `...` i izaberite **Language options**.
3. Pod sekcijom *Keyboards* kliknite na **Add a keyboard** i sa liste izaberite **Serbian (latin) - QWERTY** .
4. Staru fabričku tastaturu (*Serbian (Latin) - QWERTZ*) možete slobodno obrisati na tri tačkice kako vam ne bi smetala pri šetanju kroz jezike (`Win + Space`).

---
*Napomena: Ovaj drajver radi na nivou Windows kernela i ne troši resurse niti zahteva .NET Framework 3.5 nakon instalacije.*
